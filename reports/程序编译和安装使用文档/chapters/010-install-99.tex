\subsection{guide-docs 与 docs 对齐规则（固定）}\label{guide-docs-alignment-rules}

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  \texttt{guide\_docs/} 用于靶场负责人交付与过程记录；\texttt{docs/} 用于交付材料与最终口径。
\item
  两套文档对同一事实（IP、端口、拓扑、命令、目录）必须保持一致。
\item
  SSH 登录凭据直接保存在 \texttt{96-靶场交付说明.md} 和 \texttt{98-WSL连接服务器指南.md} 中。
\end{enumerate}
