# 靶场搭建指南（guide_docs）

本目录是你作为“靶场负责人”的交付物：在**只有一台内网 Linux 主机**的前提下，搭一个满足课程验收的**≥5 节点靶场**（中心机 + 多个客户机 + C2），并保证：

- **可部署**：一键启动/停止（建议 Docker Compose）
- **可复现**：可一键清空数据、重复演示
- **可观测**：能稳定产生 DNS/HTTP/SSH 等网络证据 + 主机日志/行为证据

> 说明：本目录只讲“靶场工程落地”。字段/接口/算法的权威口径请以 `docs/51/52/53/54` 为准。
