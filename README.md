# Attack Trace Analyzer（恶意攻击行为溯源分析系统）

本仓库用于完成《网络空间安全与信息安全课程设计》综合性题目 2：

> **基于主机日志、主机行为、网络流量的恶意攻击行为溯源分析系统设计与实现**

当前阶段以“文档先行”为主：把需求、规格、数据库/索引设计、开源项目调研等材料沉淀下来，便于团队在 3 天开发窗口内快速对齐方案与分工。

## 文档入口

- `docs/00-文档索引.md`：文档总览（建议从这里开始）
- `docs/课程设计要求文档.md`：课程/验收/提交要求（老师给的题目说明整理版）
- `docs/04-模块与数据流说明.md`：面向组员快速上手的流程讲解（从采集到串链）
- `docs/99-选型决策记录.md`：选型拍板清单（把所有可选项最终定下来）

## 目录结构（约定）

- `analyzer/`：核心分析逻辑（事件归一化、关联、攻击链重建等）
- `docs/`：课程报告相关文档与设计说明
- `docs/images/`：文档插图
